About Me

I'm a passionate software developer with a knack for solving complex problems and building efficient solutions. I have experience working with Python, Java, C/C++, JavaScript, and MATLAB and have applied my skills to projects ranging from network simulations to real-time IoT systems.Some of my notable projects include a NetRoute Simulator using Dijkstra's algorithm for shortest path calculation, an IoT-based Smart Agriculture System with real-time monitoring and environmental control, and a MATLAB syntax validator built with Python and PLY. I'm driven by curiosity, creative problem-solving, and the desire to make an impact through technology.
